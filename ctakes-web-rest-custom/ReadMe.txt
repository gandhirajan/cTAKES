Build the complete ctakes source code first before building this module.
Build this module and deploy the WAR file generated out of the build in Tomcat
Access the URL http://<HOST_NAME>:<PORT>/ctakes-web-rest/index.jsp for testing the REST service
To test using REST client like Postman, use the following URL - http://localhost:8080/ctakes-web-rest/service/analyze?pipeline=Default
Ensure to use POST method and RAW response for testing using REST client
